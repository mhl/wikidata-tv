Suggest a random TV episode to watch!

This site uses episode data from the amazing Wikidata project to suggest a random episode of a particular series you like to watch.

For lots of series, unfortunately, the data in Wikidata isn't good enough yet for this to work, but this site will suggest ways in which you can edit Wikidata to improve this - both helping this site and the world in general.

If you're interested in learning about how to help out with the Wikidata project, you could start with the Wikidata Tours
